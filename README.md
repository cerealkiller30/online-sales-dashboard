# Online Sales Dashboard (Excel)

This project analyzes an online sales dataset (Kaggle) and generates:

- Cleaned dataset
- Monthly aggregated data
- Forecast using Python (numpy polyfit)
- Interactive Excel dashboard:
  - KPI cards
  - Revenue trend line chart
  - Category-wise revenue bar chart
  - Slicers (Month, Category, Payment Method)

## Files Included
- `clean_sales_data.xlsx` — cleaned dataset
- `monthly_sales.xlsx` — monthly aggregated values
- `kpi-dashboard.xlsx` — final interactive dashboard
- `sales-kpi.ipynb` — Python analysis code

## Dataset Source
Kaggle: *<a href="https://www.kaggle.com/datasets/shreyanshverma27/online-sales-dataset-popular-marketplace-data">Online Sales Dataset – Popular Marketplace Data</a>*
